About Me

Engineer by craft, builder by instinct

Hi, I'm Nicholus Mutugi

A software engineer from Kenya, passionate about building technology that makes a difference. I specialize in full-stack development, creating everything from mobile applications to complex backend systems.

My journey in tech started with curiosity about how things work — taking apart old electronics, writing my first lines of code, and never stopping the pursuit of knowledge. Today, I work across the entire stack: React Native for mobile, modern JavaScript frameworks for web, Python and Node.js for backends, and SQL/NoSQL databases for data persistence.

When I'm not coding, you'll find me exploring machine learning concepts, contributing to open-source projects, or sharing knowledge with aspiring developers in my community.
