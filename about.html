About Me

Engineer by craft, builder by instinct

Hi, I'm Nicholus Mutugi

I’m a software engineer from Kenya, driven by a deep curiosity for how things work and a passion for building technology that makes an impact. I specialize in full-stack development, creating solutions that range from mobile and web applications to complex backend systems.

My path into tech was driven by curiosity—breaking things apart to understand them, writing code, and continuously learning. Today, I work across the stack—building mobile apps with React Native, developing modern web applications, and engineering backend systems using Python and Node.js, supported by both SQL and NoSQL databases.

Beyond coding, I explore Machine learning and Artificial Intelligence, contribute to open-source projects, and actively share knowledge within the developer community.
