# CV / Resume Downloads

Place your CV PDF file in this directory with the name:
- `Nicholus_Mutugi_CV.pdf`

## How to create your CV PDF:

1. Export your CV from your document editor (Google Docs, Word, etc.)
2. Save as PDF format
3. Name it `Nicholus_Mutugi_CV.pdf`
4. Place it in this `cv/` folder

The download button on the CV page will automatically link to this file.

## Alternative: Generate from HTML

You can also use the "Print" button on the CV page and select "Save as PDF" to generate a PDF directly from the HTML version.
